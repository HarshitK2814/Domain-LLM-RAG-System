# Domain-Specific LLM Reasoning & RAG System

## Overview
Implementation of a domain-adapted Large Language Model using parameter-efficient fine-tuning (LoRA) and Retrieval-Augmented Generation (RAG).

## Key Features
- LoRA-based fine-tuning using HuggingFace Transformers
- FAISS-backed document retrieval pipeline
- Automated factuality and hallucination evaluation
- Hyperparameter tuning experiments

## Tech Stack
Python, PyTorch, HuggingFace, FAISS
